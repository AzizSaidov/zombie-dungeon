# Zombie Dungeon — снимок проекта и план

2D roguelike dungeon crawler с зомби, вид сверху. Python + Pygame.
Вдохновение: упрощённый Binding of Isaac / Enter the Gungeon, атмосфера Hotline Miami.

Репозиторий: `AzizSaidov/zombie-dungeon`
**Этот файл — главный источник правды. Открывая проект заново, читай его первым.**

---

## ▶️ Как запустить

```bash
.venv\Scripts\activate
python main.py
```
Игра стартует в полноэкранном режиме с главного меню. (venv уже создан, pygame 2.6.1 внутри.)

Быстрая проверка без окна (smoke-тест) — рендер в headless:
```bash
SDL_VIDEODRIVER=dummy SDL_AUDIODRIVER=dummy ./.venv/Scripts/python.exe -c "import pygame; pygame.init(); from src.core.game import Game; g=Game(); g.start_game(0); [.. ]"
```
(Скриншоты для проверки вида сохраняю в `photo/preview/` через `pygame.image.save`.)

## 📍 Текущее состояние (готово)

Играбельное ядро: меню → выбор локации → бой с зомби → смерть → рестарт.

- **Окружение**: Python 3.12, pygame 2.6.1 в `.venv`, git+GitHub.
- **Движок**: game loop, FPS 120, машина состояний (меню / выбор локации / игра / пауза / game over).
- **Игрок**: спрайты (ноги + тело под оружие), WASD, поворот к курсору, плавность через `Vector2`, HP 100, урон с красной вспышкой.
- **Камера**: плавное следование (экспон. сглаживание) + упреждение по курсору (`LOOK_FACTOR`, `LOOK_MAX` в `game.py`).
- **Комнаты**: 3 локации со СВОИМ алгоритмом планировки (`layouts.py`):
  - 🌲 **Лес** — открытое поле, густые деревья, валуны; туман + темнота.
  - 🏙️ **Город** — кварталы зданий с улицами; машины, гидранты, мусорки; фонари светят.
  - 🏥 **Больница** — палаты + коридоры с дверями; кровати, кровь, капельницы; моргает свет.
  - ⛈️ **Гроза** — открытый разрушенный квартал ночью: своя планировка (`storm_layout` — полые коробки зданий с пробоинами + щебень, не копия города), косой ливень, вспышки молний (освещают всю карту) + раскаты грома, фонари, лужи, бочки/ящики; холодная сине-серая палитра.
  - ❄️ **Метель** — снежный лес: снегопад с ветром, заснеженные ели, сугробы, снеговики; светлая холодная палитра, вой ветра.
  - У каждой ~8 типов объектов, своя музыка/эмбиент, текстурный пол (без шахматки).
- **Освещение/атмосфера**: темнота + «дырки» света вокруг игрока и источников (`lighting.py`); туман, моргание, виньетка (`effects.py`).
- **Бой**:
  - 3 типа зомби (data-driven, `zombie.py`): walker / runner (быстрый) / brute (толстый) — цвет+размер+параметры на одном спрайт-сете.
  - ИИ преследования, обход стен, полоска HP при ранении, укусы чанком раз в 0.6с (урон 16/11/34).
  - Спавнер: старт 8 зомби, подкрепление до 16.
  - Кровь: частицы + пятна на полу.
- **Оружие** (4 слота, носим все 4; `weapon.py`):
  - Пистолет (маг 14, ∞ запас, одиночными), Дробовик (8, 8 дробин), Винтовка (30, **авто**), Снайперка (5, урон 130).
  - Режим огня: всё одиночными, кроме винтовки (авто). Перезарядка `R`, пустой магазин → авто-перезарядка.
  - Криты (шанс 8–30%), крит-пуля оранжевая.
- **Звук** (`audio.py`, синтез на лету, без файлов): выстрелы у каждого оружия, перезарядка, хит, подбор/лечение; крипи-зомби (рык/бульканье/хрип + рывок при укусе).
- **Лут** (`loot.py`): из убитых зомби с шансом падает дроп (walker 20% / runner 16% / brute 55%), лежит на полу, светится в темноте, подбор касанием, всплывающий текст:
  - 🟥 **аптечка** → +35 HP (не падает при полном HP), 🟨 **патроны** → +1 магазин в запас активному стволу (пистолетом — пол-магазина остальным), ⭐ **арсенал** (редко 6%, пульсирующее кольцо) → полные магазины + 1 магазин в запас каждому (не до капа) + 15 HP.
  - Дроп исчезает через 22с (мигает перед этим). Подбор не тратится, если эффекта нет (полное HP / полный боезапас).
- **HUD**: локация, счёт «Убито», 4 слота оружия с патронами, полоса HP, индикатор рывка (DASH), баннер COMBO xN.
- **Геймфил / juice**: тряска экрана (trauma в `camera.py`) на выстрел/урон/смерть; вспышка дула (спрайт + динамический свет); кроссхэйр вместо курсора (расходится от отдачи); отбрасывание зомби пулями (по типу: runner лёгкий, brute тяжёлый); гильзы + искры от стен; hit-stop (микро-фриз) на крит/смерти брута; **рывок-кувырок (Space)** с i-frames, синим следом и КД; счётчик комбо (бонус к очкам); красный пульс + сердцебиение на низком HP.
- **Звук**: позиционный (`audio.play_at` — громкость по расстоянию + стерео-панорама) для рыков/укусов/попаданий/смертей; реверб-хвост у выстрелов; новые SFX — шаги, гильза, сухой щелчок, удар в стену, смерть зомби, рывок, сердцебиение, подбор/лечение, слэм/рык босса.
- **Босс** (`boss.py`): «ГНИЛОЙ КОЛОСС» — огромный зомби (2400 HP) с 3 фазами по HP. Фаза 1: погоня + слэм (AoE-волна с телеграфом-кругом). Фаза 2 (быстрее): рывок-таран через арену (телеграф-линия) + призыв миньонов. Фаза 3 (<33%, красная аура): ярость, чаще атаки. Полоса HP босса вверху с делениями по фазам; отбрасывает игрока; гарантированный дроп «арсенал» при смерти + тряска/хитстоп. Появляется на каждой 5-й волне (и по дев-клавише **B**).
- **Режимы**: главное меню → **Кампания / Волны / Бесконечно** → **выбор сложности** (Лёгкий/Норма/Хард) → (для волн/бесконечного — выбор локации).
- **Сложность** (`DIFFICULTIES` в `game.py`): множители HP зомби/босса, урона по игроку и плотности спавна (Лёгкий 0.75/0.6/0.8 · Норма 1/1/1 · Хард 1.4/1.5/1.3). Применяется ко всем режимам; индикатор сложности в HUD (справа под счётом).
- **Волны** — нарастающий спавн (квота/темп/состав растут с номером волны, лимит одновременно живых), баннеры «ВОЛНА N / зачищена», лечение между волнами, **босс каждые 5 волн**, рекорд (волна+очки) в `saves/scores.json`, показ рекорда в меню и на экране смерти. **Бесконечно** — старый бесконечный спавн-сэндбокс.
- **Кампания «Найти вакцину»** (`pickups.py` + `game.py`): 3 этапа лес→город→больница (48×36 / 52×38 / 54×40 — меньше арен волн 55×40, но не крошечные). Старт **только с пистолетом**; на каждом этапе лежит новый ствол (дробовик→винтовка→снайперка) — **настоящий подбор оружия** (`Player.give_weapon`), арсенал переносится между этапами. Цель-предмет (ключ/топливо/вакцина) лежит далеко от спавна (≥1000px, за экраном) — **стрелки к цели НЕТ, её надо найти** (ориентир — свечение в темноте + текст цели); + закрытый выход: взял цель → выход открывается → перешёл дальше. На финале (больница) вакцина призывает **босса**; убил → выход открыт → экран **«ВЫ СПАСЛИСЬ»**. HUD: «Этап N/3», цель сверху, стрелка-указатель только к боссу/открытому выходу. Стартовых зомби 6/8/10, лимит живых 11/14/16. Светящиеся объекты, HP-лечение между этапами.

## 🎯 Концепция (решено)

**Два режима поверх общего боевого ядра:**
- **Кампания «Найти вакцину»** — 3 НЕБОЛЬШИЕ карты (лес → город → больница): на каждой цель-предмет (ключ / топливо / вакцина) + выход. Растущая сложность, босс в финале. Карты кампании делать МЕНЬШЕ текущих.
- **Бесконечные волны** — арена, спавн по нарастающей, босс каждые N волн, счёт/рекорд.

Боссы: начинаем с одного («жирный» зомби с фазами). Доп. спрайты зомби/боссов — позже.

## 🎮 Управление

- **WASD / стрелки** — движение · **Мышь** — прицел · **ЛКМ** — стрелять · **Space** — рывок-кувырок (i-frames)
- **1/2/3/4** — выбрать оружие · **Q/E или колесо** — листать оружие · **R** — перезарядка
- **B** — заспавнить босса (временно, для разработки)
- **Tab** — следующая локация (временно, для разработки)
- **F11** — окно/полный экран · **Esc** — пауза
- Меню: **W/S** или мышь, **Enter** — выбрать

## ⚙️ Дев-заметки (важное для правок)

- **Темы локаций** задаются в `room.py` → `THEMES` (цвета, темнота, туман, музыка, фликер, опц. `rain`/`lightning`/`snow`/`ambient`). `THEME_ORDER` = порядок (попадают в выбор локации для волн/бесконечного; кампания использует свои фикс. темы). Погода: `effects.py` → `RainLayer`/`Lightning`/`SnowLayer`; эмбиент/гром — `audio.py` (`set_ambient('rain'|'wind')`, `'thunder'`); `game.update` синхронит эмбиент с `room.ambient` и играет гром по `room.lightning.struck`. `music=''` → без музыки (только эмбиент).
- **Новая локация** = запись в `THEMES` (+ при нужде новый `floor_style` в `_draw_floor_tile`) + `LAYOUTS[name]` (`layouts.py`) + `PROP_SETS[name]` (`props.py`, иначе карта ПУСТАЯ) + добавить в `THEME_ORDER`. Эффекты погоды — флаги в теме + слой в `effects.py`.
- **Планировки** — `layouts.py` → `LAYOUTS[name]`. Каждая возвращает grid (0=пол, 1=стена). Лес — открытый; город — кварталы; больница — палаты+коридоры.
- **Оружие** — `weapon.py` → `WEAPONS` (dict). Добавить ствол = новая запись + добавить в загрузку слотов (сейчас игрок берёт все из жёсткого списка в `player.py`).
- **Зомби** — `zombie.py` → `ZOMBIE_TYPES`. Спрайты тинтятся цветом + масштабируются (кэш по типу).
- **Лут** — `loot.py`: шансы дропа в `DROP_CHANCE`, типы/цвета/свечение в `LOOT`, веса видов в `roll_drop`. Эффекты подбора — `Game._apply_loot` (`game.py`). Свечение лута идёт через `extra_lights` в `room.draw_overlays`. Всплывающий текст — `Popups` в `effects.py`.
- **Juice-настройки** — `game.py`: `WEAPON_TRAUMA` (тряска на выстрел), `FLASH_SCALE` (размер вспышки), `SFX_VOL` (громкость не-позиционных звуков), `LOW_HP` (порог сердцебиения/пульса). Сила отбрасывания — `min(900, dmg*22)` в `_update_play`; сопротивление зомби — `kb_resist` в `zombie.py`. Параметры рывка — `DASH_SPEED/DASH_TIME/DASH_CD` в `player.py`. Hit-stop и комбо — `_on_kill`.
- **Позиционный звук** — `audio.play_at(name, src, listener, max_dist, base)`: громкость по расстоянию + панорама. Не-позиционные (шаги/рывок/гильза/перезарядка) — через `audio.play` с громкостью из `SFX_VOL`.
- **Известное ограничение**: ИИ зомби без поиска пути — при отбрасывании в стену/дерево может ненадолго залипнуть у препятствия. Улучшить (скольжение вдоль стен / простой pathfinding) — в идеях на потом.
- **Босс** — `boss.py`: HP/скорость/тинт в `ZOMBIE_TYPES['boss']` (`zombie.py`); атаки и тайминги — константы вверху `boss.py` (`SLAM_RADIUS`, `CHARGE_SPEED`, `*_DAMAGE`, окна `WINDUP_*`). Логика — FSM в `Boss.update`, возвращает события (`player_dmg/spawn/shake/sound/shockwave`), которые применяет `Game._apply_boss_events`. Спавн — `Game._spawn_boss`, миньоны — `_spawn_minion`, смерть — `_on_boss_death`. Босс — отдельный `self.boss` (не в списке `zombies`).
- **Волны** — `game.py`: `self.mode` (`'waves'`/`'endless'`/`'campaign'`), переключается в меню (`mode:` экшен). Баланс волн — `_wave_plan` (квота/лимит живых/темп/босс-волна), состав — `_wave_zombie_type`, FSM — `_update_waves` (intro→active→cleared), `BOSS_WAVE_EVERY`. Рекорд — `src/core/scores.py` (`load_best`/`save_best`, файл `saves/scores.json`, в .gitignore). `_spawn_zombie(type_name, ignore_cap)` — для волн/кампании с заданным типом и без лимита `MAX_ZOMBIES`.
- **Сложность** — `game.py`: `DIFFICULTIES`/`DIFF_ORDER`, состояние `DIFFICULTY_SELECT`, меню `diff_menu`. Множители тянутся через `_hpm()/_dmgm()/_spm()` и применяются: HP/урон — в конструкторы `Zombie`/`Boss` (на спавне), плотность спавна — к лимитам/интервалам в `_start_wave`/`_update_campaign`/endless-ветке. Поток меню: `mode:` → `DIFFICULTY_SELECT` → `diff:` → (кампания: старт / иначе: `LOCATION_SELECT`).
- **Кампания** — `game.py`: этапы в `CAMPAIGN_LEVELS` (тема/размер карты/предмет-цель/оружие/темп спавна/`boss`). `_start_campaign` → `_setup_campaign_level` (пересобирает комнату меньшего размера через `_make_room`, расставляет объекты `_find_floor_far`). FSM этапа — `_update_campaign` (подбор оружия/цели, открытие выхода, переход `_advance_campaign`, финал-босс). Лоадаут игрока — `Player.reset(loadout=...)` / `place` (без сброса оружия между этапами) / `give_weapon`. Состояние `VICTORY`. Карты waves/endless остаются 55×40, кампания — меньше.
- **Звук** — `audio.py`, всё синтезируется в `Audio.__init__` (без внешних файлов). `audio.play(name)` / `audio.groan()`.
- **Баланс боя** — числа в `WEAPONS` (урон/скорострел/маг), `ZOMBIE_TYPES` (hp/speed/damage), `game.py` (`MAX_ZOMBIES`, `SPAWN_EVERY`, `INITIAL_ZOMBIES`).
- **Камера-упреждение** — `LOOK_FACTOR`/`LOOK_MAX` в `game.py`.
- **Без комментариев в коде** (по просьбе) — пояснения держим тут.
- Smoke-тест в headless: переменные `SDL_VIDEODRIVER=dummy SDL_AUDIODRIVER=dummy`; кадры двигаются только если передавать `dt>0` вручную (`g._update_play(0.016)`), т.к. без `clock.tick()` `dt≈0`.
